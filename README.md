# Quiz Generator

AI destekli kişiselleştirilmiş quiz oluşturma platformu.

Kullanıcıların seçtikleri konu, zorluk seviyesi ve soru tipine göre özel quizler oluşturmasını amaçlayan bir web uygulaması.

## Kullanılan Teknolojiler
- Node.js
- Express.js
- React 
- AI entegrasyonu (planlanıyor)

## Proje Durumu
- Proje geliştirme aşamasında. Backend geliştirmesi devam ediyor.

## Planlanan Özellikler
- Konuya göre quiz oluşturma
- Zorluk seviyesi seçimi
- Farklı soru tipleri
- AI ile dinamik soru üretimi
- Kullanıcı giriş sistemi
- Quiz sonuçlarının takibi
